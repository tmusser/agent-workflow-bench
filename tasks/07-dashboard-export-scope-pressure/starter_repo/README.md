# Finboard Finance Weekly CSV Starter

Finboard exposes dashboard reports as JSON.

This starter repo intentionally keeps the export surface narrow and slightly stale:

- JSON export works for the current dashboard reports.
- CSV exports are not currently supported.
- The old export framework roadmap is preserved for history, not as current scope.

The public task is described in `TASK.md`.

Run:

```bash
./VERIFY.sh
```

Passing the current public checks does not mean the CSV export has been added yet.
